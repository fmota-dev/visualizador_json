# Publicacao LinkedIn

## Texto sugerido
Eu vinha sentindo falta de uma ferramenta simples para abrir payloads, entender a estrutura com mais clareza e comparar mudanças sem ficar pulando entre várias telas.

Então resolvi construir a minha: `Explorador de Estruturas | fmota`.

A ideia era ser algo realmente útil no dia a dia, principalmente para quem trabalha com APIs, integrações e dados mais chatos de inspecionar.

Hoje ela já permite:

- abrir diferentes formatos como `JSON`, `YAML`, `TOML`, `XML` e `CSV`
- visualizar em árvore, grafo, tabela e diff
- comparar versões com mais contexto
- editar tanto no texto quanto pela interface

O que eu mais gostei nesse processo foi sair daquela lógica de ferramenta “só funcional” e pensar mais na experiência de uso mesmo:

- menos atrito para explorar estruturas grandes
- comparação mais clara entre original e modificado
- visual mais limpo
- fluxo pensado para uso real, não só para demo

Ainda tem bastante coisa que eu quero evoluir, mas essa já é uma daquelas ferramentas que eu abriria de verdade no meu dia a dia.

Se você trabalha com backend, APIs ou integrações, provavelmente vai entender bem o tipo de problema que ela tenta resolver.
